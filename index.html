mapper v1.0.1

The input for the program will be two (2) .txt files:

  • For the list of points each line of text will represent a point’s x and y coordinates separated by a comma (i.e. x, y).
  • For the adjacency matrix each line of text will represent a row of the matrix.

The first row in each file represents the first node of the graph and the last row in each file represents the last node of the graph – the nodes are name in alphabetical order from first to last starting with the letter A.
